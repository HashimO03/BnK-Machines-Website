# BnK — Full Stack Web Application

> A full-stack web application built with Java Spring Boot, featuring user authentication,
> an admin panel, and careers system.

🌐 **Live Site:** [your-domain.up.railway.app](https://bk3dprinter-production.up.railway.app/)

---

## Features

- User authentication & role-based access control
- Admin & committee management panel
- Careers system with CV upload and application tracking
- Discussion board
- Contact & enquiry management
- Light / Dark mode
- Fully responsive design

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Java 17, Spring Boot 3.4, Spring Security |
| Frontend | Thymeleaf, HTML5, CSS3, JavaScript |
| Database | MySQL, Spring Data JPA |
| Auth | BCrypt, session-based security |
| Deployment | Docker, Railway |
| Build | Gradle |
